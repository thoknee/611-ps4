# Legends: Monsters & Heroes

This is a turn based fantasy RPG implemented in Java featuring grid-based exploration, party management, tactical combat, and an extensible OOP architecture. The game recreates a classic “Heroes & Monsters” style experience with clean class design, factories, dynamic maps, and a full inventory + combat system.

## Overview
**Legends: Monsters & Heroes** is a modular role-playing game where the player controls a party of heroes exploring a dynamically generated world. The player can:

- Travel the world  
- Fight monsters  
- Shop at markets  
- Equip weapons and armor  
- Cast spells  
- Use potions  
- Manage inventories  
- Level up  
- Inspect heroes in detail  

## Gameplay Features

### Grid-Based World
- Randomized map generation  
- Accessible tiles, bushes, markets, and monster tiles  
- Battle triggers on certain tile types

### Party of Heroes
- Player controls multiple heroes  
- Each hero has unique stats and class-specific leveling behavior

### Monster Encounters
- Monsters scale to hero levels  
- Multiple types with unique strengths and weaknesses

### Turn-Based Combat
- Attack using weapons  
- Cast spells (fire, ice, lightning)  
- Use potions  
- Monsters choose attacks automatically  
- Damage calculated using hero stats, agility dodge systems, and armor mitigation

### Markets
- Buy/sell weapons, armor, spells, and potions  
- Inventory updates dynamically  
- Price and level requirements enforced

### Inventory & Equipment
- Weapons (one-handed or two-handed)  
- Armor  
- Spells  
- Potions  
- Fully supported hero inventory menu  
- Ability to inspect equipment and stats at any time
